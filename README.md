# Backend FreshCut

Este backend está construido con Spring Boot y Maven.

## Requisitos
- Java 17+
- Maven 3.8+

## Desarrollo
1. Configura `src/main/resources/application.properties`.
2. Ejecuta: `mvn spring-boot:run`.

## Pruebas
`mvn test`

## Despliegue
Genera el JAR con `mvn package` y despliega según tu entorno.
